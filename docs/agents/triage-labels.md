# Triage labels — Lumiera

| Role (mattpocock) | Label / Status line | Meaning |
|-------------------|---------------------|---------|
| `needs-triage` | `needs-triage` | Agente/mantenedor precisa avaliar |
| `needs-info` | `needs-info` | Aguardando informação do reporter |
| `ready-for-agent` | `ready-for-agent` | Especificado — AFK agent pode pegar |
| `ready-for-human` | `ready-for-human` | Requer implementação humana |
| `wontfix` | `wontfix` | Não será feito |

Em `.scratch/` issues, gravar como `Status: ready-for-agent` no topo do markdown.

GitHub Issues: aplicar labels com os mesmos nomes se existirem no repo.
